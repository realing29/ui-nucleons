Виджет модификаторов с основными свойствами:

```jsx
      <ModifiersWidget
        title='Варианты'
        items={items}
        onSelectItem={onSelectItem}
      />
```

Виджет модификаторов с ссылкой на таблицу размеров:

```jsx
      <ModifiersWidget
        title='Размер / рост'
        items={items}
        onSelectItem={onSelectItem}
        sizesTableUrl='test/url'
      />
```
