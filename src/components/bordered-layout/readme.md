Рамка только сверху:
```jsx
    <BorderedLayout top >
        <div  style={{ padding: 16 }}>
            some child
        </div>
    </BorderedLayout>
```
Рамка сверу и снизу:
```jsx
    <BorderedLayout top bottom >
        <div  style={{ padding: 16 }}>
            some child
        </div>
    </BorderedLayout>
```
Рамка только снизу:
```jsx
    <BorderedLayout bottom >
        <div  style={{ padding: 16 }}>
            some child
        </div>
    </BorderedLayout>
```
